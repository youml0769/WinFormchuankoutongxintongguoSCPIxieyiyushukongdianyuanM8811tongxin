# WinForm串口通信：通过SCPI协议与数控电源M8811通信

## 项目简介

本项目提供了一个基于WinForm的应用程序，通过串口通信实现与数控电源M8811的SCPI协议通信。用户可以通过该应用程序发送SCPI命令，控制M8811电源的输出参数，如电压、电流等。

## 功能特点

- **串口通信**：通过串口与数控电源M8811进行通信。
- **SCPI协议**：支持SCPI协议，可以发送和接收符合SCPI标准的命令。
- **参数控制**：可以设置和读取电源的电压、电流等参数。
- **实时监控**：实时显示电源的输出状态和参数。

## 使用说明

1. **配置串口**：
   - 打开应用程序后，首先配置串口参数（如波特率、数据位、停止位等），确保与M8811电源的串口设置一致。

   2. **连接电源**：
      - 点击“连接”按钮，建立与M8811电源的串口连接。

      3. **发送SCPI命令**：
         - 在命令输入框中输入SCPI命令，点击“发送”按钮，即可向电源发送控制命令。
            - 例如，设置输出电压为5V的命令为：`VOLT 5`。

            4. **读取参数**：
               - 可以通过发送读取命令（如`VOLT?`）来获取当前的电压值。

               5. **断开连接**：
                  - 使用完毕后，点击“断开”按钮，断开与电源的串口连接。

                  ## 注意事项

                  - 确保串口线连接正确，避免通信错误。
                  - 在发送命令前，确认电源处于待机或工作状态。
                  - 如果遇到通信问题，检查串口配置和电源状态，确保参数设置正确。
